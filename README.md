# twomassage

합법 방문 웰니스 마사지 서비스를 위한 Next.js App Router 사이트입니다.

## 실행

```bash
npm install
npm run dev
```

## 포함 항목

- 서비스, 요금, 예약, 지역, 신뢰센터, 후기, FAQ, 웰니스 가이드 페이지
- 페이지별 metadata, canonical, robots noindex 처리
- Organization, LocalBusiness, Service, Article, FAQPage, BreadcrumbList JSON-LD
- App Router `sitemap.ts`, `robots.ts`
- 모바일 하단 고정 상담 CTA
